# E-Commerce Sales Dashboard

## Project Overview
Analyzed 9,994 orders from Superstore dataset to identify sales trends and top products.

## Key Findings
- Total Revenue: $2,297,200
- Best Year: 2014 ($733,947)
- Top Product: Canon imageCLASS 2200 Copier ($61,599)
- Highest Category: Furniture ($741,999)

## Tools Used
- Microsoft Excel
- PivotTables
- Charts (Line, Bar, Pie)
